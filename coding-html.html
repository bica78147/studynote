코딩

(web1)코딩-html

html(hypertext markup language)

  • 웹페이지를 만드는 문법(코드)
  • 쉽고 중요함

태그(tag)

  • 웹을 컨트롤 할수 있는 문법

열리는 태그

  • 형태:(왼쪽으로 튀어나온 꺽쇠)(태그 이름)(오른쪽으로 튀어나온 꺽쇠)

닫히는 태그

  • 형태:(왼쪽으로 튀어나온 꺽쇠)(슬래시)(태그 이름)(오른쪽으로 튀어나온 꺽쇠)

시작태그의 형태만 있는 이름의 태그 종류도 있음

부모태그와 자식태그

  • 열리는 태그와 같은 이름의 닫히는 태그 사이에 다른 태그를 감쌀때 감싸고 있는(또는 더 큰 상위의)태그를 부모 태그라고 한다
  • 포함된(또는 더 작은 하위의)태그를 자식 태그라고 한다
  • 대부분은 부모 자식 관계가 되는데 태그의 종류(이름)은 별로 중요하지 않지만 몇몇 종류의 태그들은 그 관계가 고정된 태그들이 있다
  • (자식 태그가 있닥 해서 부모가 꼭 와야 하는아니고 부모가 있다고 해서 자식이 꼭 와야 하는것도 아니고 때때로 이런 태그들도 부모 자식 관계가 바뀔수도 있ㄷ..사실 잘 모르겠다)

속성(attribute)

  • 태그의 이름 또는 다른 속성 뒤 띄어쓰기를 한 뒤 써서 태그에 더 많은 의미를 부과할수 있는 html에서 중요한 문법
  • 한 태그에 속성을 하나 더 추가하는법:(띄어쓰기)(속성이름)="(속성값)"
속성 종류 css의 ".(값)"선택자의 값과 같은 class의 값이 포함된 태그에 그 선택자가 주는 효과를 받는다 값은[class="(값)"]에서 괄호 안 부분에 원하는 문자를 넣을수 있고 그 값은 띄어쓰기로 구분하여 여러 값을 넣을수 있다(그럼으로 띄어쓰기가 값의 이름엔 들어갈순 없다) id:css의 "#(값)"선택자의 값과 같은 id의 값이 포함된 태그에 그 선택자가 주는 효과를 받는다 값은[id="(값)"]에서 괄호 안 부분에 원하는 문자를 넣을수 있고 그 값은 띄어쓰기로 구분하여 여러 값을 넣을수 있다(그럼으로 띄어쓰기가 값의 이름엔 들어갈순 없다)

+태그를 의미에 맞게 사용하는 것은 여러 이유로 중요하다

html을 작성할때 맨 앞에 (왼쪽으로 튀어나온 꺽쇠)(!doctype html)(오른쪽으로 튀어나온 꺽쇠)을 쓰자 이건 태그가 아니고 어떤 버전의 html로 작성 되었는지 정보를 전달하는 특수한 명령이다

태그 종류

a(ancher)

  • 열리는 태그와 닫히는 태그 사이의 내용을 클릭했을때 지정한 페이지로 이동시킨다
  • 이미 방문한 페이지로 나타내는 링크면 보라색, 방문하지 않은거면 파란색 으로 글씨 색깔을 변경한다 하지만 css같은거로 색깔을 지정하면 그 색깔로 고정된다

body(구조 정의 태그)

  • 본문을 이 태그의 이름의 열리는 태그와 닫히는 태그로 감싸면 브라우저가 코드 내용을 쉽게 이해할수 있다

br

  • 한줄을 띄운다(줄바꿈)

font

  • 열리는 태그와 닫히는 태그 사이의 내용을 지정한 색깔로 바꾼다(html5부터 지원하지 않음)

h(1-2-3-4-5-6)

  • 열리는 태그와 닫히는 태그 사이의 내용을 본문의 제목으로 만든다
  • 열리는 태그 앞에 줄바꿈을 하고(h뒤에 숫자가 높을수록 더 많이 띄움) 열리는 태그와 닫히는 태그 사이의 내용을 h뒤에 붙은 숫자에 따라 크기를 변경한다(작을수록 커짐)그리고 진하게 표시한다

head(구조 정의 태그)

  • 본문을 설명하는 내용(본문이 아닌 부분)을 열리는 태그와 닫히는 태그로 감싸면 브라우저가 코드 내용을 쉽게 이해할수 있다

html(구조 정의 태그)

  • 이 페이지의 모든 html내용을 열리는 태그와 닫히는 태그로 감싸면 브라우저가 코드 내용을 쉽게 이해할수 있다

img(image)

  • 특정한 이미지를 웹에 표시한다(속성 필요)

li(list)(부모태그:ul,ol)

  • 열리는 태그와 닫히는 태그 사이에 있는 글을 하나의 항목으로 만든다
  • 열리는 태그와 닫히는 태그 사이에 있는 글 앞에 큰 점이 생기고 (부모태그가 ol이라면 점 대신 자신이 몇번째 항목인지를 번호로 나타낸다)내용 뒤에 1줄을 줄바꿈한다

meta(본문태그가 아님)

  • 웹 페이지에 추가 정보를 제공한다(속성 필요)

ol(ordered list)

  • 열리는 태그와 닫히는 태그 사이에 있는 항목들(li태그로 인해 항목이 됨)을 같은 리스트에 넣는다
  • 마지막 항목의 마지막 글자 뒤에 줄바꿈을 추가로 한번 하고 각각의 항목 앞에 큰 점 대신 그 항목의 번호를 표시한다

p(paragraph)

  • 열리는 태그와 닫히는 태그 사이를 한 단락으로 한다
  • 열리는 태그와 닫히는 태그 사이의 내용의 마지막 글자 뒤 줄바꿈을 2번 한다

span

  • 아무런 효과가 없다

strong

  • 열리는 태그와 닫히는 태그 사이의 글을 진하게 표시한다

style

  • 열리는 태그와 닫히는 태그 사이의 글을 css의 문법에 맞게 해석해서 표시하라고 브라우저에게 명령한다

title(본문 태그가 아님)

  • 열리는 태그와 닫히는 태그 사이의 내용을 웹 페이지의 이름으로 만든다

u(under line)

  • 열리는 태그와 닫히는 태그 사이의 글 아래에 밑줄을 긋는다

ul(unordered list)

  • 열리는 태그와 닫히는 태그 사이에 있는 항목들(li태그로 인해 항목이 된)을 같은 리스트에 넣는다
  • 닫히는 태그 뒤 줄바꿈을 추가로 한번 한다
